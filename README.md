# AI_Career_Coach
AI-powered career assistant for resume improvement, cover letter generation, and career advice using LLM.

Built using Python and Gradio that helps users improve their job applications and career decisions.

 Features
	•	💬 Career Chatbot – Ask questions about jobs and careers
	•	📝 Resume Polishing – Improve resume content based on job role
	•	📄 Cover Letter Generator – Create customized cover letters
	•	🎯 Career Advisor – Get personalized career advice

 Technologies Used
	•	Python
	•	Gradio
	•	LLM (AI models)

 How it works

The application uses AI to process user input such as job descriptions, resumes, and questions, and generates intelligent responses to assist users in their career journey.
